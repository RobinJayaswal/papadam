== README

README

This is a website created for alternative musician Papadam. Features his music and videos. Allows for admin to log in and update blog posts right from the website. Also has automated email integration that allows for fans to sign up for mailing list. Finally, a “Contact Us” section allows fans to write messages to Papadam from the website.

Built with Ruby on Rails 4.1.6, Ruby 2.1.3
